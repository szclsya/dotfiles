# i3wm and py3status Config files
To use, copy `git-i3status.conf` to `i3status.conf`.

Also, these packages are needed.
```
# From Arch Linux official repository
adobe-source-code-pro-fonts #For the status bar font
python-mpd2 #For py3status to display audio infomation
mpd #The Music Player Daemon
mpc #MPD's command line tools, used to control mpd at status bar
light #Scren brightness control 
pulseaudio #Audio backend
# From AUR
py3status #Better status 
ttf-font-awesome-4 #Problem with the official ttf-font-awesome version 5, use old version in AUR
```
